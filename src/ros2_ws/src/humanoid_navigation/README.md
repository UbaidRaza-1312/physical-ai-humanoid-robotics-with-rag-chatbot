# Humanoid Navigation Package

This package focuses on the localization, mapping, and path planning capabilities for the humanoid robot. It will integrate with ROS 2's Nav2 stack and VSLAM systems.

## Nodes

- Integration nodes for VSLAM (`rtabmap_ros` or similar).
- Navigation nodes for interfacing with Nav2.
